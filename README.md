# Android Picture Change (사진 변환)

스위치를 통해 시작할지 안할것인지를 설정하며 

시작을 할시 라디오 버튼을 통해 원하는 사진을 선택해 볼수있다. 

종료 , 처음처럼의 버튼이있으며 종료를 누를시 자동 종료되고 

처음처럼을 누를시 모든것이 초기화 되어 처음부터 다시 시작하게된다.
